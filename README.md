## 👋关于这本书

数据结构和算法是计算机科学的基础，它更多的是一种解决问题的思维。因此我建议您将它作为一本工具书使用，隔一段时间翻阅一下。由于本书侧重算法而非代码，因此您需要在开始之前阅读一下关于 Python 语法的书籍《Python 编程：从入门到实践》，这本书您也可以在我的 Github 主页中找到。
